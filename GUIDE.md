# 🚜 巨量本地推 · GitHub Pages 部署指南

先在 GitHub 上创建仓库，然后运行几条命令即可。

---

## 第一步：创建 GitHub 仓库

1. 打开 https://github.com 并登录（没有账号的话点 Sign up 注册，免费）
2. 点右上角 **+** 号 → **New repository**
3. 填写：
   - **Repository name**: `promo-analyzer`
   - **Public**（公开，免费）
   - 其他保持默认
4. 点击 **Create repository** 按钮

## 第二步：回到终端运行推送命令

在下面输入你的 GitHub 用户名，我帮你生成推送命令。

> 在输入框里 **输入 `! ` 后跟你的GitHub用户名**，例如：
> `! my-github-username`

然后我就能帮你把代码推上去并开启 Pages 了。
